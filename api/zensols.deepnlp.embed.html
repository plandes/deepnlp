zensols.deepnlp.embed.glove module¶

This library contains the definition of a class that operates like a dict to retrieve GloVE word embeddings. It also creates, stores and reads a binary representation for quick loading on start up.

class zensols.deepnlp.embed.glove.GloveWordEmbedModel(name, path, cache=True, lowercase=False, desc='6B', dimension=300, vocab_size=400000)[source]¶

Bases: zensols.deepnlp.embed.wordtext.TextWordEmbedModel

This class uses the Stanford pretrained GloVE embeddings as a dict like Python object. It loads the glove vectors from a text file and then creates a binary file that’s quick to load on subsequent uses.

An example configuration would be:

[glove_embedding]
class_name = zensols.deepnlp.embed.GloveWordEmbedModel
path = path: ${default:corpus_dir}/glove
desc = 6B
dimension = 50
Parameters

  • path (Path) – the directory path to where the unziped GloVE embedding text vector files and binary representation files subdirectory lives

  • desc (str) – the size description (i.e. 6B for the six billion word trained vectors)

  • dimension (str) – the word vector dimension

  • vocab_size (int) – vocabulary size
