Deep learning framework example: Clickbate dataset¶

Important: Please see the MNIST notebook example in the zensols.deeplearn API first, as it contains more explaination of how the framework is used.

See the saved notebook to see the output of this example.

# environemnt configuration and set up: add this (deepnlp) library to the Python path and framework entry point
from mngfac import JupyterManagerFactory
fac = JupyterManagerFactory()
mng = fac()

Print information about¶

Use the factory to create the model executor. The write method gives statistics on the data set that is configured on the executor.

In [2]:
from io import StringIO
# read the configuration from glove.conf using the same command line process to load the config and models
facade = mng.create_facade('glove_50')
from zensols.config import Writable
# set indention level for human readable (pretty print like) output
Writable.WRITABLE_INDENT_SPACE = 2
facade.write(include_model=False, include_metadata=False, include_settings=False)

Try fasttext news embeddings¶

Note that we can experiment by setting the embedings directly in the old facade. However, recreating the facade is usually better to capture the proper set up by starting fresh. All configuration is reset and reloaded so the language features are added back automatically.

In [6]:
# fasttext embeddings converge faster so lower the epoch count
facade.epochs = 30
facade = mng.create_facade('fasttext_news_300')
mng.run()
facade.persist_result()
